                                                        # 块级作用域

                                                        块代码：在es5中形同虚设

                                                        {
  var foo = "foo"
}
console.log(foo)

                                                        只有两个东西会形成作用域：

                                                        1. 全局作用域
                                                        2. 函数作用域

                                                        ES5 不支持块级作用域，但有一些技巧可以模拟块级作用域。

                                                        例如，使用立即执行函数表达式 (IIFE) 可以创建一个块级作用域：

                                                        (function() {
  var x = 10;
  // x 只在这个 IIFE 内部有效
})();

                                                        请注意，如果使用 var 声明的变量，它在声明前就已经可用，即使在其声明的代码块内。因此，使用 IIFE 是模拟块级作用域的有效方法。

                                                        此外，还可以使用构造函数或对象创建块级作用域，但这并不是推荐的方法。

                                                        总的来说，在 ES5 中，如果需要块级作用域，则可以使用 IIFE。

                                                        ES6（ECMAScript 2015）引入了块级作用域，这是一种更易于使用和更灵活的作用域。

                                                        在 ES6 中，使用 let 和 const 声明变量可以创建块级作用域：

                                                        //对let/const/function/class声明的类型有效,var没有,不过大部分浏览器为了兼容以前的代码，让function没有块级作用域
{
  let x = 10;
  // x 只在这个代码块内有效
  function bar(){}
  class Person{}
}

                                                        # 函数的剩余参数

                                                        在 JavaScript 中，剩余参数允许函数接受任意数量的参数。这些参数将被当作一个数组传递给函数。剩余参数语法在 ES6 中引入。(必须放到最后，否则会报错)

                                                        剩余参数的语法如下：

                                                        function functionName(param1, param2, ...rest) {
  // code to be executed
}

                                                        其中 ...rest 是剩余参数的标识。剩余参数将接收所有的剩余参数，并将它们存储在一个数组中。

                                                        例如：

                                                        function addNumbers(a, b, ...numbers) {
  let result = a + b;

  for (let i = 0; i < numbers.length; i++) {
    result += numbers[i];
  }

  return result;
}

let sum = addNumbers(1, 2, 3, 4, 5);
console.log(sum); // 15

                                                        在这个例子中，函数 addNumbers 可以接受任意数量的参数，并将它们存储在 numbers 数组中。然后，我们循环遍历 numbers 数组，并将每个数字相加，最后返回结果。

                                                        # 展开语法

                                                        JavaScript 展开语法是 ECMAScript 6 中引入的新语法，用于将数组或对象的所有元素展开到新数组或对象中。

                                                        对于数组，展开语法可以使用三个点 (...)，如下所示：

                                                        const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];
const arr3 = [...arr1, ...arr2];
console.log(arr3);  // 输出 [1, 2, 3, 4, 5, 6]

                                                        对于对象，展开语法也可以使用三个点 (...)，如下所示：

                                                        const obj1 = {a: 1, b: 2};
const obj2 = {c: 3, d: 4};
const obj3 = {...obj1, ...obj2};
const obj = {...obj2,f:5}//构建对象字面量（浅拷贝）
console.log(obj3);  // 输出 {a: 1, b: 2, c: 3, d: 4}

                                                        展开语法还可以用于函数参数，以实现将数组或对象作为多个参数传递给函数：

                                                        function sum(a, b, c) {
  return a + b + c;
}
const arr = [1, 2, 3];
console.log(sum(...arr));  // 输出 6

                                                        # reflect的作用

                                                        他是一个对象，字面意思是反射，可简化 Proxy 的创建。

                                                        前面所讲过的内部方法，例如 [[Get]] 和 [[Set]] 等，都只是规范性的，不能直接调用。

                                                        Reflect 对象使调用这些内部方法成为了可能。它的方法是内部方法的最小包装。

                                                        open in new window

                                                        proxy还是对原来的对象做操作，用reflec：

                                                        let user = {
  name: "John",
};

user = new Proxy(user, {
  get(target, prop, receiver) {
    alert(`GET ${prop}`);
    return Reflect.get(target, prop, receiver); // (1)
  },
  set(target, prop, val, receiver) {
    alert(`SET ${prop}=${val}`);
    return Reflect.set(target, prop, val, receiver); // (2)
  }
});

let name = user.name; // 显示 "GET name"
user.name = "Pete"; // 显示 "SET name=Pete"

                                                        receiver —— 如果我们源对象（obj）有setter，getter的访问器属性，那么可以通过receiver来改变里面的this

                                                        const person = {
  name: 'John',
  age: 30
};

const handler = {
  get(target, property, receiver) {
    console.log(receiver); // 输出：{ custom: 'property' }
    return target[property];
  }
};

const proxyPerson = new Proxy(person, handler);

const customObj = { custom: 'property' };
proxyPerson.name; // 这里的receiver将是全局对象，因为没有传递第三个参数
proxyPerson.name.call(customObj); // 这里的receiver将是customObj对象

                                                        Reflect.construct()是JavaScript Reflect API中的一个方法，它提供了一种动态创建对象的方式，可以在运行时以类似于new关键字的方式构造一个新的对象。

                                                        class Person {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }
}

const args = ['John', 30];

const person = Reflect.construct(Person, args);

console.log(person); // 输出：Person { name: 'John', age: 30 }
